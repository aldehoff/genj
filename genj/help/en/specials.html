Special Information

This section contains information about special features of GenJ. The novice user doesn't have to use them right away. Stick to entering your genealogic information and explore all the standard features.

Once you start to work with others, exchanging files, you will want to merge files or check the compliancy of a file. There will probably be more special Tools like that - for now have a look here to learn about the existing ones.

GenJ can run Reports on a file - but there isn't a big and sophisticated selection yet. To learn about writing your own read on here - please make sure to share any reports you come up with.

To localize GenJ you can either edit one of the language-jar's content by hand (have a look into genj_en.zip) or you can run a little ui-tool Resedit
