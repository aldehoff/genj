Tools

There are currently two tools hooked up in the main menubar.

Verify Gedcom

This tool verifies a Gedcom file - first choose the file to check for compliancy. Then choose from the given options which should be checked (might not be changeable for now).

Merge Gedcoms

This tool merges two Gedcom files - first choose the file to be read from. Then choose the file to perform the merge into. After clicking OK those files will be merged and you can see the results in the latter file.

You have to be very careful when merging files because overlapping id information will be matched when GenJ tries to identify information that matches!
