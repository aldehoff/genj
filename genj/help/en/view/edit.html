Views - Edit View

As discussed in the section about Gedcom one entity consists of a hierarchy of properties and their values. After opening this view it will follow any selection that you might make in any view in GenJ. The hierarchy of properties of the selected entity will be displayed including symbols that represent each property, the Tags and values.

Explore the tree of properties and use the scrollbars at the right and bottom if necessary. After clicking on a property an editor for that type of property will appear in the bottom half if applicable. Change values there if desired.

Some of the properties contain references to other entities (e.g. HUSB keeps a reference of a family's husband and FAMS the reference of a spouse's family). Click twice on such a node and the current entity will be switched to that target.

The plus-button in the toolbar (see below) allows to add new properties. An additional property is added as a sub-property of the currently selected property. A dialog will present all Gedcom compliant sub-properties - choose one or more and they're inserted. You can also add properties through custom Tags in the same dialog but it's advised to do so sparingly as to not divert too far from the standard.

Don't forget the Context Menu to perform complex actions on properties and the entity, too. Try a right-click on one of the displayed nodes and discover what you can do.

Toolbar

  1. add a property to selected property (continues in dialog)
  2. remove currently selected property
  3. move currently selected property up
  4. move currently selected property down
  5. return to the last entity shown before the current one
  6. toggle sticky (stick with current entity - don't follow selection in other view)
  7. close view

Settings

There are no settings for this view at this point.
