Control Center

After starting GenJ a window is opened showing a list of Gedcom files. From here you can open views on individual files by clicking on one of the visible buttons:

Opens/Creates a file
Opens a Table View on the selected file,
Opens a Tree View on the selected file,
Opens a Timeline View on the selected file,
Opens an (Property-)Edit View on the selected file,
Opens a Report View on the selected file,
Opens the Navigator View,
Opens a Settings dialog.
Please click on one of the hyperlinks above to learn more about that feature.

The menu offers items to open/save/close a file and end the program. Two tools are hooked up to the menu which allow merging of two files and verification of a file.

The first four buttons at the bottom allow to create entitities. From left to right : Individual, Family, Media and Note. The button on the right opens a dialog to delete an entity.
