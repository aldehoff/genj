Ansichten - Report Ansicht

Ein Report ist eine Java Klasse, welche eine Schnittstelle  implementiert zu GenJ. Reports sind im ./reports Verzeichnis abzulegen und werden dynamisch durch GenJ geladen. Sobald Sie die Report Ansicht geöffnet haben, wird eine Liste der verfügbaren Reports gezeigt. Sie können aus der Liste auswählen und eine Beschreibung des Reports sowie seines Autors wird rechts angezeigt werden.

Die Auswahlknöpfe erlauben den Report zu starten, ihn anzuhalten, die Ansicht zu schließen, das Ergebnis des Reports zu speichern oder den Reports erneut zu laden, falls er geändert wurde. Beachten Sie, dass alle Reports auf die Gedcom Datei zugreifen, welche zum Zeitpunkt der Auswahl der Report Ansicht geöffnet war.

Abhängig vom Typ des Reports, springt diese Ansicht zur Ergebnis Karteikarte nachdem der Report gestartet wurde. Ein Report kann entweder den Typ Konsole oder Fenster besitzen. Der letztgenannte Typ erstellt eine einfache Ausgabe, welche innerhalb der Karteikarte dargestellt wird (kann später abgespeichert werde). Beim erstgenannten Typ erscheinen angepasste Dialogfenster, ohne dass das Ergebnis im Standardausgabeformat erstellt wird.
