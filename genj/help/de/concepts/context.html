Arbeiten mit dem Kontextmenü

Es ist wichtig, dass Sie während der Arbeit mit GenJ ausprobieren, die Anzeige von Kontextmenüs auszulösen. Ein Rechtsklick (Bei Ihre Plattform kann eine andere Maus Aktion erforderlich sein, um das Kontextmenü auszulösen) auf der angezeigten Information in einer der Ansichten wird Ihnen zeigen, welche Aktion Sie dort auslösen können.

Das Kontextmenü ist immer in der gleichen Form strukturiert: an der ersten Stelle erscheint ein Untermenü, welches zeigt auf welches Element Sie geklickt haben und dann folgend weitere Untermenüs, welche den Kontext erweitern.

Im folgendem Beispiel wird das Kontextmenü für die Namen Eigenschaft angezeigt. Das erste Untermenü zeigt die Eigenschaft selbst und die möglichen Aktion auf dieser Eigenschaft, das zweite Untermenü zeigt die Instanz (hier ein Individuum), der die Eigenschaft zugeordnet ist, das dritte Untermenü zeigt die Datei (Gedcom), der die Instanz zugeordnet ist.

Das Untermenü für das Individuum 'I008' ermöglicht die folgenden Aktionen:

  • Kind hinzufügen - ein Kind (und eine Familie wenn nötig) für I008
  • Partner hinzufügen – ein Partner (und Familie wenn notwendig) für I008
  • Geschwister hinzufügen – ein Geschwister (und Eltern wenn notwendig) für I008
  • Bibliothek hinzufügen – ein externe Bibliothek mit Informationen zu I008
  • Multimediaobjekt hinzufügen- ein Objekt mit binären Daten
  • Notiz hinzufügen – eine Notiz zu I008
  • Übermittler hinzufügen- eine Person welche Informationen übermittelt hat zu I008
  • Quelle hinzufügen – die Quelle der Informationen über I008
  • Assoziation hinzufügen- eine Beziehung zu einem anderen Individuum
  • Löschen – Entfernen der Instanz I008

Das nächste Beispiel zeigt ein Kontextmenü für eine Eigenschaft aber diesmal werden die Aktionen für die Eigenschaft selbst angezeigt (erstes Untermenü).

Dies sind die möglichen Aktionen:

  • Notiz hinzufügen – diese Notiz kann Daten enthalten, die speziell die Geburt näher beschreiben
  • Quelle hinzufügen – diese Quelle kann festhalten, woher die Informationen zu dieser Geburt herstammen
  • Multimediaobjekt hinzufügen – dieses Objekt kann binäre Daten abspeichern, die diese Geburt betreffen
  • Löschen –Informationen über die Geburt (Eigenschaft BIRT) komplett löschen

Zum Schluss ein Beispiel welches demonstriert, wie der Inhalt der Kontextmenüs dynamisch erstellt wird. Die folgende Logik wird angewendet, wenn das Menü bei einem Rechtsklick auf einem Knoten in der Baum Ansicht generiert wird:

  • Der Kontext auf den geklickt wurde ist keine Eigenschaft sondern eine Instanz (hier Individuum I008) somit ist das erste Untermenü das für ein Individuum
  • Die Ansicht in der das Kontextmenü aufgerufen wurde - die Baum Ansicht – hat einen Ansichtspezifischen Eintrag zum Untermenü hinzugefügt, die Aktion Bookmark hinzufügen.

Die Aktionen, die im geöffneten Untermenü angezeigt werden, erlauben das Hinzufügen einer neuen Instanz in der Datei example.ged. Es wird dazu geraten, "existierende Information verbunden wachsen zu lassen" statt neue alleinstehende (nicht durch Beziehungen verknüpfte) Instanzen wie Individuen oder Familien hinzuzufügen. Um verbundene/ referenzierte Instanzen zu erzeugen, benutzen Sie das Untermenü für Eigenschaften oder Instanzen wie oben beschrieben..
