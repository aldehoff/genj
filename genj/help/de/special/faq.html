F Um was geht es bei GenJ?
A GenealogyJ (kurz GenJ) ist ein Viewer und Editor für genealogische Daten im Gedcom Format.

F Gibt es ein Diskussionsforum bezüglich GenJ ?
A Ja, bitte gehen Sie zu unserem Forum @ genj.sourceforge.net

F Was bedeutet Gedcom?
A Bitte lesen Sie die Erklärung hier.

F Wie kann ich Bilder in der Baum Ansicht anzeigen lassen?
A Um Bilder für jede Person in der Baum Ansicht anzuzeigen, fügen Sie jedem Individuum eine Multimedia Eigenschaft (OBJE) hinzu. Lassen Sie ein Individuum in der Editier Ansicht anzeigen, wählen Sie die in der Hierarchie die oberste Eigenschaft INDI aus, klicken Sie auf den '+' Auswahlknopf und wählen Sie OBJE zum Hinzufügen aus. Lassen Sie die Untereigenschaft FILE auf eine Datei in Ihrem Computer Dateisystem verweisen (Auswahlknopf '>>') und die Bilder werden aus der Baum Ansicht heraus angezeigt werden (Die Ansicht sucht automatisch nach dem INDI:OBJE:FILE).

F Kann ich die von GenJ benutzte Sprache ändern?
A Bitte laden Sie das passende Sprachen Package (genj_ll-x.y.zip) für GenJ (z.B. genj_fr-2.0.zip) von der Projekt Homepage herunter und folgen Sie den Anweisungen, um es zu installieren. GenJ wird automatisch versuchen, die Sprache Ihres Betriebssystems zu verwenden. Um die verwendete Sprache explizit zu ändern (das Sprachen Package muss manuell installiert werden), wählen Sie den Menüeintrag Hilfe|Über und wählen Sie den Menüpunkt Look&Feel an. Dann wählen Sie die gewünschte Sprache aus.

F Wie kann ich einen Zentimeter auch als einen auf meinem Bildschirm darstellen?
A GenJ benutzt Masse der realen Welt für verschiedene Komponenten (z.B. in der Baum Ansicht). Um die Längen von Linien und Größen von Boxen korrekt auf Ihrem Bildschirm anzuzeigen, müssen Sie GenJ Ihre Bildschirmauflösung mitteilen. Um die Auflösung explizit anzupassen, wählen Sie den Menüeintrag Hilfe|Über und dann den Menüpunkt Look&Feel. Benutzen Sie ein Lineal, um das Lineal auf dem Bildschirm entsprechend anzupassen.

F Kann ich die langweilige graue Darstellung ändern?
A GenJ verwendet das Swing Toolkit zur Darstellung von Informationen. Dieses benutzt Grau als Default – es wird dort Metal genannt. GenJ erlaubt die Änderung des Look&Feel. Bitte laden Sie das Look&Feel Package (genj_lnf-x.y.zip) von der Projekt Homepage und folgen Sie den Installationsanweisungen. Nach dem Neustart von GenJ, wählen Sie bitte den Menüeintrag Hilfe|Über und dann Look&Feel. Nun können Sie das Look&Feel und optional das Theme auswählen.

F Wie kann ich bestehende Datensätze miteinander verbinden?
A Ich habe eine existierende Person und ich möchte es als Kind einer anderen schon existierenden Person einfügen. Die einzige verfügbare Option, scheint das Hinzufügen einer *neuen* Kind/Eltern Beziehung zu einem existierende Datensatz zu sein, aber man kann scheinbar keine *existierende* Person einer Kind/Eltern Beziehung hinzufügen. Wie kann ich das tun? Sie können das existierenden Elternteil auswählen (oder zur Eltern Familie des 'existierenden' Kindes). Wählen Sie dann "Kind Hinzufügen" aus dem Kontextmenü und in dem angezeigten Dialog wählen Sie die gewünschte Person oben aus dem Drop-Down Auswahl. Auf diese Weise wird keine neue Person als Kind erzeugt sondern nur eine Beziehung zwischen den schon existierenden.
