F Um was geht es bei GenJ?
A GenealogyJ (kurz GenJ) ist ein Anzeige- und Bearbeitungsprogramm für genealogische Daten im Gedcom-Format.

F Gibt es ein Diskussionsforum bezüglich GenJ ?
A Ja, bitte gehen Sie zu unserem Forum @ genj.sourceforge.net

F Was bedeutet Gedcom?
A Bitte lesen Sie die Erklärung hier.

F Wie kann ich Bilder in der Baum-Ansicht anzeigen lassen?
A Um Bilder für jede Person in der Baum-Ansicht anzuzeigen, fügen Sie jedem Individuum eine Multimedia-Eigenschaft (OBJE) hinzu. Lassen Sie ein Individuum in der Editier-Ansicht anzeigen, wählen Sie die in der Hierarchie oberste Eigenschaft INDI aus, klicken Sie auf den '+' Auswahlknopf und wählen Sie OBJE zum Hinzufügen aus. Lassen Sie die Untereigenschaft FILE auf eine Datei in Ihrem Computer-Dateisystem verweisen (Auswahlknopf '>>') und die Bilder werden aus der Baum-Ansicht heraus angezeigt werden (Die Ansicht sucht automatisch nach dem INDI:OBJE:FILE).

F Kann ich die von GenJ benutzte Sprache ändern?
A Bitte laden Sie das passende Sprach-Paket (genj_ll-x.y.zip) für GenJ (z.B. genj_fr-2.0.zip) von der Projekt-Homepage herunter und folgen Sie den Anweisungen, um es zu installieren. GenJ wird automatisch versuchen, die Sprache Ihres Betriebssystems zu verwenden. Um die verwendete Sprache explizit zu ändern (das Sprach-Paket muss manuell installiert werden), wählen Sie den Menüeintrag Hilfe|Über und den Menüpunkt Look&Feel. Nun wählen Sie die gewünschte Sprache aus.

F Wie kann ich einen Zentimeter maßstabsgetreu auf meinem Bildschirm darstellen?
A GenJ benutzt Maße der realen Welt für verschiedene Komponenten (z.B. in der Baum-Ansicht). Um die Längen von Linien und Größen von Boxen korrekt auf Ihrem Bildschirm anzuzeigen, müssen Sie GenJ Ihre Bildschirmauflösung mitteilen. Um die Auflösung explizit anzupassen, wählen Sie den Menüeintrag Hilfe|Über und dann den Menüpunkt Look&Feel. Benutzen Sie ein Lineal, um das Lineal auf dem Bildschirm entsprechend anzupassen.

F Kann ich die langweilige graue Darstellung ändern?
AGenJ verwendet das Swing-Toolkit zur Darstellung von Informationen und benutzt standardmäßig grau – Metal genannt. GenJ erlaubt die Änderung des Look&Feel. Bitte laden Sie das Look&Feel-Paket (genj_lnf-x.y.zip) von der Projekt-Homepage und folgen Sie den Installationsanweisungen. Nach dem Neustart von GenJ wählen Sie bitte den Menüeintrag Hilfe|Über und dann Look&Feel. Nun können Sie das Look&Feel und optional das Theme auswählen.

F Ich habe eine vorhandene Person und möchte sie als Kind einer anderen schon existierenden Person einfügen. Die einzige verfügbare Option scheint das Hinzufügen einer *neuen* Kind/Eltern-Beziehung zu einem existierende Datensatz zu sein, aber man kann scheinbar keine *existierende* Person einer Kind/Eltern Beziehung hinzufügen. Wie kann ich das tun?
A Sie können den existierenden Elternteil (oder die Eltern-Familie des 'existierenden' Kindes) auswählen. Wählen Sie dann "Kind hinzufügen" aus dem Kontextmenü und in dem angezeigten Dialog wählen Sie oben die gewünschte Person in der DropDown-Auswahl. Auf diese Weise wird keine neue Person als Kind angelegt sondern nur eine Beziehung zwischen den schon existierenden erzeugt.

F Wie gebe ich das ADOP(tion)-Kennzeichen richtig ein?
A Fügen Sie der Adoptiv-Familie das adoptierte Kind hinzu: öffnen Sie FAMilien-Instanz, wählen Sie "Familie => Kind hinzufügen" und selektieren Sie nun die adoptierte und hinzuzufügende Person aus dem DropDown-Menü. Dadurch wird die INDI-Instanz der Person um eine FAMC-Eigenschaft erweitert. Bearbeiten Sie nun die INDI-Instanz der Person und fügen Sie eine ADOP-Eigenschaft hinzu. Benutzen Sie jetzt Ziehen und Ablegen (Drag'n Drop) oder Ausschneiden/Einfügen (in 2.2+MDI-Version), um die FAMC-Eigenschaft unter INDI in die ADOP-Eigenschaft zu verschieben. Wählen Sie die eingeschlossene und gerade verschobene FAMC-Eigenschaft und fügen Sie ihr einen ADOP-Eintrag hinzu, um kenntlich zu machen, wer das Kind adoptiert hat. Wechseln Sie zur gerade erstellten ADOP-Eigenschaft und es erscheint eine neue Option im Bearbeitungs-Fenster, um die Person auszuwählen, von der das Kind adoptiert wurde: (HUSB=Ehemann oder mälicher Partner, WIFE=Ehefrau oder weiblicher Partner, BOTH=beide Ehegatten).

F Warum wechseln Größe und Auflösung von Bildern während der Programmbenutzung?
A Bei der Aufarbeitung der Ausgabe spielen Vorlagen eine Rolle. Abhängig davon, wieviel in einem Kasten angezeigt werden muss, kann der Platz für ein Bild begrenzt sein. Es wird dann verkleinert, um genügend Platz für Text zu haben. Wenn genug Platz ist, werden die Original-Bildgröße verwendet (Pixel-Größen *und* Auflösung). Darum kann die Darstellung variieren.
