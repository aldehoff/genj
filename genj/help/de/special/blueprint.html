Vorlagen

Betrachten von Gedcom Daten in einer strukturierten hierarchischen Form mit Editier Ansicht oder Tabellen Ansicht ist eine Sache. Erzeugen der Informationen über eine Instanz in einer statischen 2-dimensionalen Form ist eine andere. Beide, sowohl die Baum Ansicht als auch die Instanz Ansicht tun gerade dieses. Sie erzeugen die Darstellung von Individuen and Instanzen in einem begrenztem rechteckigen Raum auf Ihrem Bildschirm oder Papierausdruck.

Die zugrundeliegende Technologie, die GenJ hierfür benutzt, ist HTML. Die Anweisungen, wie eine Instanz darzustellen ist, werden bestimmt durch die sogenannten Vorlagen. Sie bestehen aus Standard HTML mit einer Ergänzung durch das neue HTML Element "prop".

Hier ist ein Beispiel: 

 
      <b><prop path=INDI></b>
      <prop path=INDI:NAME>
      born <prop path=INDI:BIRT:DATE>
      in <prop path=INDI:BIRT:PLAC>,
      lived in <prop path=INDI:RESI:ADDR:CITY>
      and worked as a <prop path=INDI:OCCU>.

welches folgende Darstellung ergibt:

I008 Meier, Nils born 25 May 1970 in Rendsburg, lived in San Francisco and worked as a Software Engineer.

Wie Sie sehen können, enthält das "path" Argument zu "prop" ein Kennzeichenpfad, welcher bestimmt was innerhalb des Textflusses darzustellen ist. Bei diesem Beispiel werden die Werte der Eigenschaften wie folgt aufgelöst;

Pfad=...

Ergebnis

Individuum's ...

INDI:BIRT:DATE

INDI:BIRT:PLACE

Geburtsort

INDI:RESI:ADDR:CITY

San Francisco

Durch die Verwendung von verschiedenen Vorlagen kann die gleiche Information auf unterschiedliche Weise dargestellt werden, da unterschiedliche Informationsmengen, unterschiedliche Layouts und Farben ausgewählt werden können:

In Ansichten, die Vorlagen benutzen, können diese über ihre Eigenschaften ausgewählt und editiert werden. Vordefinierte Vorlagen können nicht verändert werden, aber durch den Benutzer definierte können durch ihn angepasst werden (Vorlagen werden global und nicht auf Ebene einzelner Ansichten verwaltet).

Bitte beachten Sie: Dies ist eine Funktion für fortgeschrittene Benutzer, die Sie nicht unbedingt erlernen müssen. Die Standardvorlagen sollten für 80% der Benutzer ausreichend sein.
