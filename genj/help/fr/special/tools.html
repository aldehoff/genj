Les Outils

Actuellement, deux outils sont disponibles à partir du menu principal de GenJ.

Vérification d'un fichier Gedcom

Cet outil vérifie un fichier Gedcom - Tout d'abord, choisissez un fichier afin de vérifier sa conformité. Ensuite choisissez à partir des options fournies, celles qui doivent être cochées (pour le moment cette dernière possibilité n'est pas activée).



Fusion de fichiers Gedcoms

Cet outil vous permet de fusionner deux fichiers Gedcom - Tout d'abord choisissez le fichier à partir duquel il convient de lire les données. Ensuite choisissez le fichier dans lequel la fusion va s'effectuer. Une fois ces deux choix faits, cliquez sur OK pour lancer la fusion. Une fois celle-ci terminée vous pourrez voir le résultat dans ce dernier fichier.
Soyez trés prudent quand vous fusionnez des fichiers car si GenJ identifie des données qui correspondent, les informations qui se chevauchent sur l'identité des personnes seront appariées!
