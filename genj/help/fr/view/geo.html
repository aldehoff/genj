Vues - La Localisation Géographique

Cette fenêtre va vous permettre de localiser les gens qui sont présents dans votre arbre généalogique.
Une fois que vous aurez importé les cartes correspondantes (voir plus loin dans les paramètres pour la procédure), vous pourrez en cliquant sur le nom d'une ville qui apparaît sur la carte, voir dans la fenêtre de droite la liste se centrer sur ladite ville, et en cliquant sur le signe "+" à côté du nom de cette même ville, visualiser tous les évènements qui se sont passés dans cette localité.
A l'inverse, si vous cliquez sur le nom d'une ville dans la liste déroulante, la carte va se centrer sur cette ville et un cercle rouge vous permettra de mieux voir où elle est.

Maintenant, vous remarquerez trés vite, que certaines localités, sont recensées comme étant "inconnu" dans la liste. C'est normal, d'une part la base n'est pas complète, et d'autre part, un certain nombre de villages n'existent plus, ou des noms ont pu changer. Ne vous effrayez pas. Cette fenêtre dispose d'un moyen pour compléter, modifier la base de données. Une chose est à noter, toutes les modifications sont enregistrées dans un fichier externe à la base de données, donc vous ne risquez pas de les perdre, si vous téléchargez plus tard une nouvelle mouture de la base de données plus à jour.
Donc pour en revenir aux modifications qu'on peut y apporter, il vous suffit de cliquer sur le bouton "Modifier" qui se trouve en bas de la liste déroulante.
Une nouvelle fenêtre va s'ouvrir. Là plusieurs possibilités ; D'une part vous pouvez voir plusieurs fois le nom de la ville qui vous intéresse, mais ce nom existe dans plusieurs départements, régions, voir pays. Choisissez celui qui vous concerne. De cette manière vous ne risquez pas de voir votre arbre se centrer sur Paris au Texas (USA), mais bien à l'endroit qu'on connaît en France si jamais vous aviez chargé et la carte de France, et la carte du Texas (USA) (un petit sourire n'a jamais fait de mal dans un document d'aide, non). D'autre part, il est possible que votre localité n'existe pas dans la base ; il vous suffit alors de rajouter les coordonnées géographiques (latitude et longitude) et de cliquez sur "Mémoriser les Coordonnées"

Dernier détail, les signes "+" et "-" qui se trouvent sous la liste des localités, vous permettent d'expanser ou de concentrer toutes les villes de la liste. Essayez, vous comprendrez facilement de quoi il retourne.


Barre d'Outils

  1. Choisir une nouvelle Carte.
  2. Revenir à la taille initiale de la Carte.
  3. Activer/Désactiver le mode Zoom.
  4. Ouvrir la fenêtre des Paramétrages.
  5. Fermer la fenêtre de Localisation Géographique.

Paramètrages

Les paramètres de cette vue sont regroupés en deux actions successives :
  1. D'une part, vous devez commencer par importer une base de données. Donc vous devez cliquer sur "Importer", et choisir dans le menu déroulant, le pays concerné par votre fichier gedcom. Il est possible que vous ayez plusieurs pays, donc vous devez faire cette importation pour chacun d'eux. N'oubliez pas qu'il vous faut une connection internet.
  2. D'autre part, une fois que cela est fait vous pouvez monter ou descendre (voir effacer) la base de données qui présente le plus d'intérêt pour votre généalogie. Le fait de mettre un fichier en premier permettra au programme de trouver immédiatement les données dont il a besoin plutôt que d'avoir à scanner toute la base.
    Bien entendu, une fois que vous avez fini, cliquez sur "Appliquer", puis "Fermer".
