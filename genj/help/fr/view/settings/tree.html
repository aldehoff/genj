Les Paramètres d'Affichage de l'Arbre Généalogique

Vous pouvez changer énormement de choses pour adapter à vos besoins la manière dont l'affichage de l'arbre généalogique va se faire. Les paramètres sont regroupés autour de trois onglets décrits ci-aprés.

N'oubliez pas d'appuyer sur "Appliquer" quand vous aurez fini!

Propriétés

Ici vous allez spécifier quelles propriétés vous souhaitez voir s'afficher dans les cadres de l'arbre généalogique. Tout d'abord, sélectionnez un type dans le menu déroulant (ex. Individus). Ensuite cochez toutes les propriétés qui vous intéressent dans l'arborescence située en dessous. Enfin réarrangez la manière dont vous voulez que ces propriétés soient affichées, dans le cadre du bas - c'est un petit peu délicat parce que vous pouvez redimensionner et repositionner ces champs à la pixel prés. En déplaçant la réglette sur le côté droit, vous pouvez activer une grille pour vous aider dans ce positionnement.

Options

Choisissez un niveau de zoom en cliquant sur l'échelle. Vous pouvez aussi activer/désactiver un certain nombre d'options :
- afficher un ombrage,
- afficher les icones de propriété,
- Appliquer le zoom aux images (attention cela utilise pas mal de ressources système surtout quand vous zoomez et que l'arbre est affiché en arborescence horizontale ou verticale),
- afficher des dates abrégées.
Enfin un choix de fontes (type et taille) vous est offert.

Signets

Un signet est un raccourci vers un endroit que l'on souhaite voir comme origine de l'arbre. Sélectionnez une entité dans n'importe quelle fenêtre de visualisation (la première ligne nous indique qu'il pointe bien sur cette entité - ex. si c'est une personne que l'on a choisie, vous verrez quelque chose du genre I006:Doe, John) et donnez un nom à ce signet dans la zone de saisie. Ensuite ajoutez le à la liste des signets de cette manière il apparaîtra dans le menu déroulant des signets de la fenêtre de visualisation de l'arbre généalogique, en bas à droite. Vous pouvez également effacer et réorganiser les signets en les déplaçant vers le haut ou vers le bas, comme vous le souhaitez.
