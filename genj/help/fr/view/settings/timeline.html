Les Paramètres de l'Affichage Chronologique

Le principal réglage consiste à sélectionner quels événements vous voulez voir s'afficher dans cette fenêtre de visualisation chronologique - tous les types d'événements sont représentés dans la sélection qui vous est proposée.

Les autres réglages possibles sont :

  •  la faculté de changer le niveau de zoom.
  •  décider si oui ou non vous souhaitez voir s'afficher les étiquettes (abbréviations) des propriétés ou juste leurs icones,
  •  ajouter les dates associés aux événements
  •  activer ou non l'affichage de la grille.

N'oubliez pas d'appuyer sur "Appliquer" quand vous aurez fini!
