Table View

This view shows all entities (of the same type) in a tabular fashion. You can scroll up/down and select entities. A selection is notified to the Edit View which will then show that entity's properties. Click on one of the column headers and the whole list will be sorted by that column - click once again and the sorting is reversed.

Individual columns represent paths to a property of an entity's property-tree. The column-headers show that path - e.g. INDI:BIRT:DATE stands for the date of the birthday-property of an individual. You'll learn more about those Tag Paths in the section about the Settings View and Report Writing.

The toolbar at the lower/right offers additional functionality:

  1. The buttons with entities' symbols choose to show either individuals, families, medias or notes
  2. The rightmost button opens the Settings View for this view
