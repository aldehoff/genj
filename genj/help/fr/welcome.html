Bonjour sur GenJ - Genealogy J

GenealogyJ (GenJ) est un logiciel libre ; vous pouvez le redistribuer et/ou le modifier en respectant les termes de la licence GNU General Public Licence telle qu'elle est publiée par la Free Software Foundation.

En ce qui concerne les derniers développements, et/ou si vous êtes intéressé par ce projet et désirez aider, vous pouvez visiter le site principal de GenJ; Vérifiez la page de cette aide consacrée à l'Equipe.


Si vous venez d'upgrader de la version 1.6 de GenJ, vous devrez vérifier la page de l'aide relative aux changements intervenus, car certains répertoires, notamment celui qui contient les rapports, ont été renommés

Pour ouvrir un fichier Gedcom... cliquez sur l'icone ou sélectionnez Fichier > Ouvrir.
Pour ouvrir la visualisation de la Table... cliquez sur l'icone.
Pour afficher l'Arbre... cliquez sur l'icone.
Pour avoir une vue chronologique... cliquez sur l'icone.
Pour éditer les Propriétés... cliquez sur l'icone.
Pour accéder aux Rapports... cliquez sur l'icone.
Pour ouvrir la fenêtre des Paramètres... cliquez sur l'icone.
