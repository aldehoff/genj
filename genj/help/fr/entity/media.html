Média

Un Média (auquel on associe l'étiquette OBJE) est une entité qui regroupe toutes les données multimédia auxquelles vous pourriez penser. GenJ accepte les fichiers images et essaiera d'afficher les informations dans n'importe quels types de visualisations (vues). Quand vous créez un média en cliquant sur vous pouvez spécifier un fichier en ouvrant un navigateur. Les données de ce fichier sont incorporées dans votre fichier Gedcom (encodées en Base64). Pour éviter de trop faire grossir votre fichier Gedcom, il vous est possible de ne pas choisir un fichier. A la place vous pouvez simplement ajouter une propriété FILE à l'entité qui par la suite va pointer sur un fichier localisé sur votre disque. Ce fichier sera externe à l'application et devra donc être acheminé séparemment.

Par défaut, les Médias ne sont pas liés aux autres entités. Aprés avoir été créée, vous pouvez ajouter une propriété OBJE (oui, OBJE est l'étiquette pour une entité média mais aussi l'étiquette pour une propriété média) à une entité quelconque et la lier à un média en entrant son id et en appuyant sur lien.

Un média, d'aprés la norme Gedcom, posséde des propriétés par défaut - voici deux exemples basés sur l'incorporation de données ou faisant référence à un fichier externe. 

@M004@ OBJE
 TITL Taken from myimage.gif
 FORM gif
 BLOB 585 Raw Bytes

@M005@ OBJE
 TITL Pointing to local file
 FORM jpg
 FILE myimage.jpg
